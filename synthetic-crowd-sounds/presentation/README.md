# My presentation

## Resume
Here is the resume of the project:

This project focuses on the parametric generation of synthetic crowd sounds for neuroscience and experimental psychology research. Crowd vocalizations are widely used to study emotion perception, multisensory integration, and social behavior, but To our knowledge, there is few standardized and reproducible database dedicated to emotional crowd sounds available on open data sources.

The objective of this project is to create controllable synthetic crowd vocalizations such as “wooo,” “ouh,” and “boo” without using real human recordings. Using Python-based audio synthesis, this project focuses on the creation of a standardized database of synthetic crowd vocalizations for use in neuroscience and experimental psychology research.

The project also emphasizes reproducibility and open science practices through the use of GitHub, DataLad, and structured audio-generation workflows.The final deliverables will include reproducible Python scripts, synthetic audio datasets and WAV stimuli.
